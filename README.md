# TravelApp 


## Techs
Project is created with:
* ReactJs
* MapBox
* Nodejs
* Mongodb


[Backend Repo](https://github.com/myamann/travel-app-api)


## Setup
To run this project, install it locally using npm:

```

 # Run Frontend 
 npm start
 
```
